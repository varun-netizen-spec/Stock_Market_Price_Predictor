# 📈 Stock Market Price Predictor

## 📂 Dataset
- Tesla stock price data (2010–2017)  

## 🛠️ Models
- Logistic Regression  
- Support Vector Machine (SVM)  
- XGBoost Classifier  

## 📊 Result
- Logistic Regression: ~54% AUC  
- SVM: ~44% AUC  
- XGBoost: ~57% AUC (overfitting)
